### Nano ID

功能：小巧的、安全的、URL 友好的、独特的 JavaScript 字符串 唯一 ID 生成器

官网：https://zelark.github.io/nano-id-cc/

使用教程：

- 安装：`npm i nanoid`
- `ES6`引入：`import { nanoid } from 'nanoid'`
- 使用：`const str = nanoid()`
